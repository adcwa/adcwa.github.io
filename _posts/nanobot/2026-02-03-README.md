## 💡 使用建议

### 如何阅读
1. **第一次**：快速浏览，了解大致内容
2. **第二次**：仔细阅读，配合源码理解
3. **第三次**：实践操作，验证理解

### 如何实践
1. 安装 nanobot：`pip install -e .`
2. 初始化配置：`nanobot onboard`
3. 边学边做：每学一个概念就动手试试

### 如何求助
- 查看源码：最权威的答案
- 查看测试：了解使用方式
- 社区讨论：Discord / GitHub Issues
